iniciar o git: git init 
Ver status: git status
Atualizar o arquivo no git: git add .
Antes de enviar o arquivo fazer comentario com: git commit -m "Comentario"
Enviar o arquivo: git push
Ver historico das atualizações: git reflog
Alterar o arquivo para versao anterior ou outra apresentada no historico: git reset --hard digitar a id(primeira sequencia de numeros)
